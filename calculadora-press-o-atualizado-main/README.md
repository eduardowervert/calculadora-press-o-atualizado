# Conversor de Pressão

Uma aplicação web simples para converter unidades de pressão.

## Funcionalidades

- Conversão entre Pa, kPa, bar, atm, psi, mmHg, Torr, inHg
- Interface responsiva
- Botão para limpar entrada

## Como usar

1. Digite o valor
2. Selecione unidade de origem
3. Selecione unidade de destino
4. Veja o resultado
5. Use o botão Limpar para resetar
